3. In MANUAL mode:
   - Set database config via /api/config/manual/database
   - Set backup config via /api/config/manual/backup
   - Set S3 config via /api/config/manual/s3
4. Switch back to ENV mode to use .env file
5. Manual config persists in memory until:
   - App restart
   - Explicit reset via /api/config/reset
